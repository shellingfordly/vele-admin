# vue3-element-plus-admin

一个基于 vue3, vite2, element-plus, vuex, vue-router, typescript 的 admin 骨架

## Preview

[浏览地址](https://shellingfordly.github.io/vue3-admin)

## Download

```
git clone https://github.com/shellingfordly/vue3-admin.git
```

## Install

```
yarn
```

## Run

```
yarn dev
```

## Build

```
yarn build
```
